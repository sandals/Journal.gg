# Journal.GG

`Journal.gg` is a web application built to help you record your progress in
League of Legend's ranked matchmaking.

Setup and Install coming soon.

```javascript
#config/secrets.js

module.exports = {
  'riot_api_key': 'YOUR_API_KEY_HERE'
};
```
